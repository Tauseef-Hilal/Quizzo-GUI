# Quizzo (GUI)

A basic GUI based quiz game written in `Python` and `PyQt6`.


## Installation
* Download the repository.
* Run **`install.py`** to install dependencies.
* Head to the **`Fonts`** folder to **install the required fonts.**


## Depends On
*playsound==1.3.0*<br>
*pyobjc==8.1*<br>
*PyQt6==6.2.2*<br>
*PyQt6-Qt6==6.2.2*<br>
*PyQt6-sip==13.2.0*
